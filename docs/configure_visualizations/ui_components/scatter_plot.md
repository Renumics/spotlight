# Scatter Plot

The Scatter Plot is similar to the [similarity map](similarity_map.md) with one big difference. It displays only two columns and places the points by the values of those two columns.

## Placement

Simply select `x-Axis` and `y-Axis` from the dropdowns in settings. The points will be placed accordingly.
