# MNIST (tiny)

A tiny subset of the **MNIST database of handwritten digits** (30 samples),
provided as sample data.

## Contents

- `0.png` – `29.png`: individual handwritten-digit images (28×28), converted to PNG.
- `mnist-tiny.csv`: table referencing the images with their `label` and a generated
  `embedding` for each sample.
